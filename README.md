# 打卡记录后台管理系统

**使用React+Koa做的一个考勤打卡后台管理查看系统**

目前使用到的技术栈

*前端*
1. React
2. Vite
3. TypeScript
4. WindiCSS
5. antd
6. echarts

*后端*
1. TypeScript
2. TypeORM
3. Koa
